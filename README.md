## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/QuizApp.git
```

#### Go to the project directory

```bash
  cd QuizApp
```

#### Install QuizApp with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
